## 🧰 Troubleshooting

### The app does not open

Try these steps:

- Make sure the download finished
- Check that you extracted the `.zip` file if there was one
- Right-click the file and choose Run as administrator
- Restart your PC and try again

### Windows shows a security prompt

This can happen with downloaded apps.

- Choose the option that lets you continue
- Confirm that you downloaded the file from the release page
- Run the file again after the prompt closes

### The app opens and then closes

Try:

- Downloading the file again
- Extracting the full release folder
- Checking that your Windows version is up to date
- Making sure no files are missing from the folder

### The app cannot connect

Try:

- Checking your internet connection
- Confirming your Claude or Anthropic details
- Closing and reopening the app
- Trying again after a minute
